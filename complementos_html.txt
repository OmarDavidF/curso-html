IP : identificador único de un página web, representa la dirección donde está el ordenador 
que contiene esa página web. cada sitio web tiene una ip única.
Dominio : el nombre asosiado a la IP que utilizamos para solicitar recursos, en nuestro
caso un sitio web.
DNS : su principal función es traducir el nombre de dominio a su identificador único.
sitio web : es un conjunto de uno a barios recuros web alojados en un mismo dominio.
servidor web : es un ordenador cuyo único objetivo es servir recursos web.
hosting : es el almacenamiento del servidor web. El disco duro donde el sevidor guarda sus 
recursos.
petición : es la acción de pedir recusrsos al servidor.
